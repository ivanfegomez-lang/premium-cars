# 🚗 Sistema de Renta de Vehículos

## 📌 Descripción
Aplicación full stack para la gestión de reservas de vehículos.

## 🛠 Tecnologías
- Angular
- Node.js
- Express
- MongoDB
- JWT

## 🚀 Funcionalidades
- Registro y login de usuarios
- Autenticación con JWT
- Listado de vehículos
- Reserva de vehículos
- Visualización de reservas

## ⚙️ Ejecución

### Backend
cd backend  
npm install  
npm start  

### Frontend
cd frontend  
npm install  
ng serve
